# Feng Construction Tools Updates

Bu klasordeki dosyalar public `Feng-Construction-Tools-Updates` reposuna kopyalanir.

- `version.json`: Programin guncelleme kontrolu icin son surum ve indirme adresi.
- `license_status.json`: Lisanslarin online active/trial/suspended/revoked durumlari.

Kaynak kod private kalir. Public tarafta sadece bu manifest dosyalari ve release asset olarak exe bulunur.
